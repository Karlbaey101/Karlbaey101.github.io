新评论系统上线！🚀（以及其他的技术更新）

最近给自己的博客折腾了点新东西，还是很开心的。下面就把这些更新写在这篇博文里吧。

新评论系统

众所周知，我的评论系统最早用的是 utterances，但是体验并不好：评论一多起来，评论输入框就会出现在页面最下方。我本来想发 issue，但是发现已经有不少人反映过这个问题了 Place new comment box above comments #513：

然后发现 icarus 模板原生支持 giscus 评论系统，我一用就喜欢上了：评论框置顶、评论表态跟回复功能。这简直就是我见过最最理想的评论系统。这里有张效果图：

现在忙着给以前托管在 issues 的评论补档，不过还是很开心的。

RSS 订阅源和 follow.it 订阅

这两个技术性调整都是为了方便喜欢我博客的朋友们及时获取订阅的，具体操作见下面：

RSS 订阅源

关于 RSS 订阅源，参考这篇文章：《知道RSS的人越少，我就越希望它能被人知道！》——阿虚同学。

考虑到我一直积极维护的互联网精神（也就是自由、开放、合作、共享的精神），我还是给自己的博客开了 RSS 订阅源。你可以按我头像下的符号或者这里来获得 RSS 订阅源。而关于如何使用本站的 RSS，请参考我的这篇博客。

follow.it 订阅

相比起 RSS 订阅，follow.it 会方便许多。左侧有一个输入框：

只要输入你的电邮并且在你的收件箱里激活，就可以通过 follow.it 提供的订阅收取我的更新通知了！它与 RSS 订阅有相似之处，不同的是，RSS 订阅更简洁明了而 follow.it 只需要查收邮件。

萌国备案

其实这是很早之前的事了，有朋友推荐我去一些人多的地方混个脸熟，于是就找到了萌社区并且注册了备案，这是我在萌社区的个人资料。但愿能招揽些人来看我的网站吧。
